WHERE TO GET IT (STEP-BY-STEP)
  1. Open Google Cloud Console: console.cloud.google.com
  2. Create/select a project: project selector
  3. Enable YouTube Data API v3 (Library): API Library
  4. Create API key (Credentials): Credentials
  5. Restrict key (recommended): in Credentials → select key → set Application restrictions and API restrictions.
